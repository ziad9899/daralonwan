# دار العنوان | Daralonwan

موقع عقاري عربي فاخر — نموذج احترافي يعرض فلل وشقق وأراضٍ مختارة في المملكة العربية السعودية، مع تجربة بحث وفلترة سهلة وتواصل مباشر عبر الواتساب.

🌐 **العرض المباشر:** https://ziad9899.github.io/daralonwan/

---

## التقنيات المستخدمة

- **React 18** + **TypeScript**
- **Vite** كأداة بناء
- **Tailwind CSS** للتصميم
- **React Router** للتنقل
- **Zustand** لإدارة الحالة
- **lucide-react** للأيقونات
- **framer-motion** للحركات

## المميزات

- صفحة رئيسية فاخرة مع شريط بحث في الـ Hero
- قائمة عقارات مع فلاتر متقدمة (المدينة، النوع، الميزانية، الغرف)
- صفحة تفاصيل العقار مع معرض صور وخريطة وهمية
- تسجيل/دخول محلي يحفظ في LocalStorage
- نموذج تواصل + زر واتساب دائم
- اتجاه RTL كامل وتصميم Mobile-First

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح: http://localhost:5173

## البناء للإنتاج

```bash
npm run build
npm run preview
```

## الهيكلة

```
src/
├── components/   # المكونات المشتركة
├── pages/        # صفحات الموقع
├── data/         # بيانات العقارات
├── store/        # Zustand store للمصادقة المحلية
└── lib/          # دوال مساعدة
```
